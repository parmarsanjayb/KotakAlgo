# 🖥️ Office Setup & Deployment Guide / ઓફિસથી પ્રોગ્રામ બદલવાની ગાઈડ

આ દસ્તાવેજ તમને સમજાવશે કે ઓફિસના કોમ્પ્યુટર પરથી આ અલ્ગો ટ્રેડિંગ પ્રોગ્રામ કેવી રીતે ચેન્જ કરવો અને કેવી રીતે સર્વર પર અપડેટ મોકલવું.

---

## 🛠️ Step 1: Copy Project Folder to Office PC (ફોલ્ડર કોપી કરો)
તમારા હાલના કોમ્પ્યુટરમાંથી આખું ફોલ્ડર `C:\xampp\KotakAlgo` ઓફિસના કોમ્પ્યુટરમાં ટ્રાન્સફર કરો. 

**ટ્રાન્સફર કરવાની રીતો:**
1. **USB પેનડ્રાઈવ દ્વારા**: આખું ફોલ્ડર પેનડ્રાઈવમાં કોપી કરીને ઓફિસના PC માં પેસ્ટ કરો (પ્રિફર્ડ પાથ: `C:\xampp\KotakAlgo`).
2. **Cloud Storage દ્વારા**: Google Drive, OneDrive, અથવા Dropbox પર આખું ફોલ્ડર અપલોડ કરી ઓફિસ PC પર ડાઉનલોડ કરો.
3. **GitHub દ્વારા (ટેકનિકલ)**: આ કોડને Private GitHub રેપોઝીટરીમાં પુશ કરીને ઓફિસના PC પર કલોન (Clone) કરો.

> [!IMPORTANT]
> ખાતરી કરો કે ખાનગી કી ફાઇલ (`LightsailDefaultKey-ap-south-1.pem` અથવા `.pem` ફાઇલ) પ્રોજેક્ટ ફોલ્ડરની અંદર જ હાજર છે. તેના વગર કોડ સર્વર પર અપલોડ નહીં થાય.

---

## ✍️ Step 2: Make Changes to Code (કોડ ચેન્જ કરો)
ઓફિસના કોમ્પ્યુટરમાં ફાઇલો બદલવા માટે:
- જો તમારે UI બદલવી હોય, તો `templates/dashboard.html` ફાઇલ ઓપન કરીને ચેન્જ કરો.
- જો બોટનું ટ્રેડિંગ લોજિક બદલવું હોય, તો `app.py` ઓપન કરીને ફેરફાર કરો.
- ફેરફાર કરવા માટે તમે **VS Code**, **Notepad++**, અથવા સાદા **Notepad** નો ઉપયોગ કરી શકો છો.
- *નોંધ: ઓફિસ પીસી પર લોકલ ટેસ્ટિંગ કરવા માટે જ Python ઇન્સ્ટોલ કરવાની જરૂર છે. જો ફક્ત કોડ બદલીને સીધો સર્વર પર જ મોકલવો હોય, તો ઓફિસ પીસી પર Python ઇન્સ્ટોલ કરવું પણ જરૂરી નથી!*

---

## 🚀 Step 3: One-Click Deploy (એક ક્લિકમાં સર્વર અપડેટ કરો)
મેં સર્વર અપલોડ પ્રક્રિયાને ૧૦૦% ઓટોમેટિક બનાવી દીધી છે. હવે તમારે કોઈ મેન્યુઅલ SSH કમાન્ડ ટાઈપ કરવાની જરૂર નથી.

1. ફાઇલોમાં ચેન્જ સેવ કર્યા પછી, ફક્ત `deploy_aws.bat` ફાઇલ પર **Double-Click** કરો.
2. આ સ્ક્રિપ્ટ ઓટોમેટિકલી:
   - તમારા ફેરફારોને ઝિપ (`KotakAlgo.zip`) કરશે.
   - ફાઇલોને સુરક્ષિત રીતે AWS સર્વર પર અપલોડ કરશે.
   - સર્વર પર ઓટોમેટિકલી લૉગિન કરીને ફાઇલો એક્સટ્રેક્ટ કરશે.
   - **અલ્ગો બોટ સર્વિસને રીસ્ટાર્ટ કરશે** જેથી તમારો નવો કોડ તરત જ લાઈવ થઈ જાય.
3. બ્રાઉઝરમાં `http://13.205.8.34` (અથવા તમારું ડોમેન) ઓપન કરો અને ચેક કરો.

## ❓ Frequently Asked Questions (વારંવાર પૂછાતા પ્રશ્નો)

### Q1: શું મારે ઓફિસ કમ્પ્યુટરમાં કોઈ સોફ્ટવેર ઇન્સ્ટોલ કરવાની જરૂર છે?
**ના.** વિન્ડોઝ ૧૦ અને ૧૧ માં ફાઇલ અપલોડ કરવા માટેના ટૂલ્સ (`ssh` અને `scp`) પહેલેથી જ ઇન્સ્ટોલ કરેલા આવે છે. જો તમે ફાઈલો બદલીને સીધા સર્વર પર મોકલો છો, તો Python ઇન્સ્ટોલ કરવાની પણ જરૂર નથી.

### Q2: જો 'Permission Denied' કે કી (Key) ની ભૂલ આવે તો શું કરવું?
તમારી પેનડ્રાઈવ કે ક્લાઉડમાંથી ટ્રાન્સફર કરતી વખતે ઘણીવાર કી ફાઈલના પરમિશન બદલાઈ જાય છે. `deploy_aws.bat` રન કરશો એટલે તે ઓટોમેટિકલી કી ફાઈલની વિન્ડોઝ પરમિશન પણ રીસેટ કરી દેશે, જેથી કોઈ ભૂલ નહીં આવે.
